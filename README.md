# node-vue-ui-website

> A Node-Vue.js project by full-stack

## Setup

``` bash
git clone https://github.com/lq782655835/node-vue-ui-website.git
cd node-vue-ui-website && npm install
npm run dev
```

## Technology

### Fed

* Vue.js
* Vue-Router
* [yi-ui](https://github.com/lq782655835/yiui)
* Axios
* Webpack4

### Back

* Node.js
* Express + Plugins
* Mongoose
